## Resolution

This issue is addressed in the following kernel updates:

- Red Hat Enterprise Linux  6.10.z-kernel-2.6.32-754.9.1.el6viaRHBA-2018:3763Workaround:
- Boot the system from kernels earlier thankernel-2.6.32-754.6.3.el6.

Red Hat Enterprise Linux  6.10.z-kernel-2.6.32-754.9.1.el6viaRHBA-2018:3763

`kernel-2.6.32-754.9.1.el6`Workaround:

`kernel-2.6.32-754.6.3.el6.`
## Root Cause

The current BUG is a regression of a patch which was introduced inkernel-2.6.32-754.6.3.el6itself. So, kernels earlier than2.6.32-754.6.3.el6don't have this issue.
