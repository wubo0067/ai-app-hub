## Resolution

- The system crashed due to deadlock which was caused as the module 'event_collector_1_9_8988'. Please contact the vendor for further investigation.The deadlock was caused as the module 'event_collector_1_9_8988' intercepts the enqueue_task and tried to allocate memory in it.As the kswapd0 was not able to continue due to deadlock situation, the memory cache was not able to be flushed out as well.

- The deadlock was caused as the module 'event_collector_1_9_8988' intercepts the enqueue_task and tried to allocate memory in it.
- As the kswapd0 was not able to continue due to deadlock situation, the memory cache was not able to be flushed out as well.
