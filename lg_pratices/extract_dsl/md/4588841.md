#### Red Hat Enterprise Linux 7

`kernel-3.10.0-1127.el7`
#### Red Hat Enterprise Linux 7.7

`kernel-3.10.0-1062.12.1.el7`
#### Red Hat Enterprise Linux 7.6

- Upgrade tokernel-3.10.0-957.43.1.el7fromErrata RHSA-2020:0179or later

`kernel-3.10.0-957.43.1.el7`

`kernel-3.10.0-862.46.1.el7`
## Root Cause

- Issue is found with the file routines for reading/proc/pagetypeinfowhen sosreport command is executed.
- Not only sosreport but also some other tools that call pagetypeinfo_showfree() to print out the free pages could invoke this sort of issue.
- Following bug is raises with this issue

Bug 1757943 - Hard lockup in free_one_page()->_raw_spin_lock() because sosreport command is reading from /proc/pagetypeinfo
