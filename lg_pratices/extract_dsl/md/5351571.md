## Environment

- Red Hat Enterprise Linux (RHEL)


## Issue

Theccissflashtool causes a kernel panic by triggering a hard lockup watchdog in the middle of applying a firmware update.

`ccissflash`
## Resolution

Please consider contacting relevant software vendor for possible assistance. The "ccissflash" tool is a 3rd-party one and is not supported by Red Hat directly.


## Root Cause

The "ccissflash" tool is performing a firmware update with interrupts disabled in userspace, which triggers a hardware watchdog after certain period.
