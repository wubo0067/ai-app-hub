## Resolution

- Engage hardware vendor to run hardware diagnostics and check for any faulty hardware.


## Root Cause

- It can be seen that many CPUs, despiteNOTperforming operations that would likely result in intense loop execution in the kernel code, are unable to properly handle timer interrupts for a very long time indicating faulty hardware as an underlying issue.
