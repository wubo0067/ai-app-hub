## Resolution

- Engage the respective hardware vendor to run the hardware diagnostics and replace the faulty CPU.


## Root Cause

- Vmcore analysis showed that there was an issue with CPU cache coherency
- In this particular example:CPU cache coherency issue with CPU 6 which does not see the latest MULTI_STOP_DISABLE_IRQ state.Hence, it does not process the stop work and effectively blocks all the other CPUs.
