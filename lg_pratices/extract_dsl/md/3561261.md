## Resolution

- Kindly use Red Hat Enterprise Linux 7.4 and 7.5 provided mpt3sas modules.
- Please engage hardware vendor and investigate the issue of the third party module.


## Root Cause

- System crashed due to a hard lockup encountered in SCSI command completion code path.
