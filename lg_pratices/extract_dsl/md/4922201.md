## Resolution


### Red Hat Enterprise Linux 7

- Update tokernel-3.10.0-1160.el7fromErrata RHSA-2020:4060or later.

`kernel-3.10.0-1160.el7`
## Root Cause

Task writing to a cgroup’s cpu.rt_runtime_us tunable parameter file gets tasklist_lock and loops in tg_rt_schedulable for an excessive amount of time causing a hard lockup.
