## Resolution

This issue is fixed in:

- Red Hat Enterprise Linux 8.7 viaRHSA-2022:7683
- Red Hat Enterprise Linux 8.6.z viaRHSA-2022:7110

Red Hat Enterprise Linux 8.7 viaRHSA-2022:7683

Red Hat Enterprise Linux 8.6.z viaRHSA-2022:7110


## Root Cause

This is a deadlock bug introduced upstream by:

- scsi: lpfc: Remove use of kmalloc() in trace event logging

This was pulled into RHEL 8.6 with thelpfcupdate onBug 1989265.

`lpfc`There are somelpfcbufixes in 8.6 onBug 2034276but not which fix this.

`lpfc`The bug is resolved upstream by:

- scsi: lpfc: Move cfg_log_verbose check before calling lpfc_dmp_dbg()
