## Environment

- Red Hat Enterprise Linux (RHEL) 6
- cbsensorkernel module

`cbsensor`
## Issue

The kernel experiences hard lockup while thecbsensorkernel module removal is in progress.

`cbsensor`
## Resolution

Thecbsensorkernel module is a 3rd-party one hence not supported by Red Hat directly. Please consider contacting relevant software vendor for further assistance.

`cbsensor`
## Root Cause

The following event sequence seems to take place:

- thecbsensorkernel module removal is initiated
- during the removal thecbsensorkernel module logging routine tried to access its already cleaned structures, which leads to a deadlock

`cbsensor``cbsensor`
