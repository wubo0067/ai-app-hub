### Red Hat Lightspeed can detect this issue


## Environment

- Red Hat Enterprise Linux release 8.7 (Ootpa)
- kernel-4.18.0-425.10.1.el8_7.x86_64 or higher
- Oracle: Out-of-tree (O) module: [oracleoks]


## Issue

- After updating the kernel from version4.18.0-425.3.1.el8to version4.18.0-425.10.1.el8_7or higher and booting into a new kernel, the machine hangs with soft lockup and rcu_sched CPU stall warning. Specifically, the lockups start when Oracle Grid Infrastructure is started. Rebooting into the old kernel version4.18.0-425.3.1.el8allows Oracle Grid Infrastructure to start without a hang.
