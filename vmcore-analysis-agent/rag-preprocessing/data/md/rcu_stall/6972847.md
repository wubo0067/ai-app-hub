## Environment

- RHEL7
- nvidia-460.73.01


## Issue

- System got a NMI waiting for a CPU which is stuck at RIP: _nv018537rm+591


## Resolution

- Engage with 3rd party module(nvidia) vendor for troubleshooting.


## Root Cause

- The last instruction is jmp which asks to move to the location of itself. CPU can not move forward due to this buggy instruction.
